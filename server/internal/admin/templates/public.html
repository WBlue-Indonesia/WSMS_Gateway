Apa ini? / What is this?

Gateway SMS internal WBlue. Bukan layanan SMS massal untuk umum — dibuat untuk pengiriman volume rendah, opt-in, dari SIM milik sendiri.

WBlue's internal SMS gateway. Not a public bulk-SMS service — built for low-volume, opt-in sending from our own SIMs.

Untuk siapa? / Who is it for?

  • Tim internal WBlue WBlue internal teams
  • Klien / aplikasi terdaftar via API key Registered clients / apps via API key
  • Operator armada ponsel Phone-fleet operators

Cara akses / How to access

  • Operator & admin → panel /admin
  • Integrasi API → minta API key ke admin Integrations → request an API key from an admin
  • Ponsel pengirim → enrollment token dari panel Sender phones → pair with an enrollment token from the console

Keamanan / Security

Setiap panggilan API pakai bearer key (disimpan ter-hash), webhook & request ditandatangani HMAC, akses panel berbasis peran dan diaudit.

Every API call uses a hashed bearer key; webhooks and requests are HMAC-signed; console access is role-based and audited.
